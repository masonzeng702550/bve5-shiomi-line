================================================================
  汐見湾鉄道 汐見線  (Shiomiwan Railway - Shiomi Line)
  BVE Trainsim 5 / 6 用 路線データ
================================================================

■ 概要
  海の街「汐見浜」から霧深い温泉郷「霧山温泉」までを結ぶ、
  全長 27.4 km・12駅の地方私鉄路線（架空）。
  一度の乗車で「海・都市・高架・地下・大鉄橋・桜並木・水田・
  里山・渓谷・温泉街」という十の風景を楽しめるように設計しています。

■ 路線諸元
  区間      : 汐見浜 〜 霧山温泉
  営業キロ  : 27.4 km
  駅数      : 12
  軌間      : 1,067 mm
  電化      : 直流 1,500 V
  最高速度  : 95 km/h（山岳区間 65 km/h）
  保安装置  : ATS-Sn
  運転時分  : 約 42 分（各駅停車・ワンマン）

■ 動作環境
  BVE Trainsim 5.7 以降 / BVE Trainsim 6
  （マップ書式 BveTs Map 2.02, 文字コード UTF-8）

■ 導入方法
  1. 本フォルダを BVE の Scenarios フォルダ配下へ配置。
  2. 車両は任意のフリー3両編成をご用意のうえ、
     scenario/shiomi_day.txt の Vehicle 行を書き換えてください。
  3. BVE のシナリオ選択画面から「汐見線 普通 霧山温泉行」を選択。

■ フォルダ構成
  scenario/           シナリオファイル
  Shiomi/map/         マップ本体（風景帯ごとに include 分割）
  Shiomi/station/     停車場リスト（時刻表）
  Shiomi/structure/   ストラクチャーリスト
  Shiomi/signal/      信号現示リスト
  Shiomi/sound/       サウンドリスト
  Shiomi/objects/     3Dモデル・テクスチャ（風景帯別）
