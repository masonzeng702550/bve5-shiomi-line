# Changelog

本プロジェクトの主な変更点。日付は制作進行の目安です。

## [1.1.0] — 開発中

### Added
- 夜行ダイヤ（`scenario/shiomi_night.txt`）— `env_night.txt` による
  光源・背景・霧・運転台照度の夜景オーバーライド。
- 雨天ダイヤ（`scenario/shiomi_rain.txt`）— 灰色の濃霧、粘着係数の低下、
  雨音を重ねた上級者向けシナリオ。
- 快速「しおかぜ」（`scenario/shiomi_rapid.txt`）— 5駅通過の
  停車場リスト `stations_rapid.txt` を用いた第2ダイヤ。
- 夜背景ストラクチャー 5種、雨音サウンドを各リストに追加。
- リポジトリ向け `README.md`。

### Notes
- 夜・雨は昼マップの線形・構造物を流用し、環境系コマンドのみを
  最終 include で上書きする方式（構造物の複製なし）。

## [1.0.0] — 初版

### Added
- 汐見線 各駅停車・昼ダイヤ（`scenario/shiomi_day.txt`）。
- 全長 27.4 km・12 駅・十の風景帯のマップ本体（`Shiomi/map/`）。
  海辺の都市／海岸線／市街高架／地下／大鉄橋／桜並木／水田／里山／
  渓谷（33‰・トンネル5本・鋼橋3本）／温泉街。
- 停車場リスト（時刻表）、ストラクチャーリスト、信号現示リスト、
  サウンドリスト。
- 4現示自動閉塞（単線）・ATS-Sn 地上子（ロング／直下／速度照査）。
- 交換駅3駅での対向列車。
